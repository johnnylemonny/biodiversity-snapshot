# Contributing to NatureNode

Thank you for your interest in contributing to NatureNode! I welcome contributions from nature enthusiasts, developers, and researchers alike.

## 🌈 How to Contribute

### Reporting Bugs
- Use the **GitHub Issues** tab to report bugs.
- Provide a clear description of the issue and steps to reproduce it.
- Include information about your environment (browser, OS, device).

### Suggesting Enhancements
- Open a GitHub Issue with the tag `enhancement`.
- Describe the feature you'd like to see and why it would be valuable for the project.

### Pull Requests
1. **Fork the repository** and create your branch from `main`.
2. **Install dependencies** using `pnpm install`.
3. **Make your changes**. Ensure your code follows the existing style and is properly linted.
4. **Test your changes** locally using `pnpm dev`.
5. **Submit a Pull Request** with a clear description of what you've changed.

## 💻 Technical Guidelines

- **Style**: We use Tailwind CSS v4 for styling. Please stick to the established design system and OKLCH color model.
- **Types**: This is a TypeScript project. Ensure all new code is properly typed.
- **Linting**: All PRs must pass the Super-Linter and CodeQL checks. Run `pnpm lint` locally before pushing.
- **Icons**: Use [Lucide React](https://lucide.dev/icons/) for all interface icons.

## 🌿 Ethical Considerations

NatureNode is built to promote environmental awareness and conservation. We expect all contributors to respect nature and follow ethical guidelines when conducting field research or providing data:
- **Do not disturb wildlife** while taking photos.
- **Respect private property** and protected areas.
- **Accuracy matters**: Ensure any data or descriptions provided are scientifically grounded.
